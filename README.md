# CSstroeFE个人主页

### 运行项目
```
// 安装全局node-dev
npm install node-dev -g 

// 进入项目根目录下
cd CStore_FE

npm install

// 修改nodeDev批处理文件
修改node_project_nodedev.bat文件
E: // 指定工程所在盘符
cd E:\Company\Code\cstroe_FE\bin // 指定工程www.js文件所在路径
node-dev www

// 双击运行node_project_nodedev.bat批处理文件（只有安装了全局的node-dev才能运行起来）

// 浏览器中运行http://localhost:10086即可访问
```
